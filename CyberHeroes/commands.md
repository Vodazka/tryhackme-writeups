#### gobuster
`gobuster - Directory/file & DNS busting tool written in Go`
```
gobuster dir -u http://<Target IP> -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt
```
> dir - the classic directory brute-forcing mode.

>-u, --url string 

> -w, --wordlist string
